# Library Observer
A C++ program to implement simmplified version of Observer design pattern that observe changes in library (checking books in/out)

## Getting Started

### To Run
1. ```make```
2. ```./lib-ob``` or ```valgrind --leak-check=yes ./lib-ob``` to check for memory leaks

### Contains
```
  - Book.cc
  - Book.h
  - Control.cc
  - Control.h
  - Library.cc
  - Library.h
  - List.cc
  - List.h
  - Logger.cc
  - Logger.h
  - main.cc
  - Makefile
  - View.cc
  - View.h
```
